HSS & CSS Reference Guide

Tags Meaning
<html> .... </html> Hyper Text Markup Language
<head> .... </head> The head, or prologue, of the HTML document
<body> .... </body> All the other content in the HTML document
<title> .... </title> The title of the document
<p> .... </p> Paragraph
<br> .... </br> Break
<div> .... </div> It is a division or part of an HTML document
<h1> .... </h1> Headings: H1, H2,... tags are used to create titles.
<img> .... </img> For images in document
<ol> .... </ol> Ordered list
<ul> .... </ul> Unordered list
<li> .... </li> A list item in bulleted (ordered list)
